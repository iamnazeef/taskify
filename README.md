# Taskify

Taskify is a TO-DO App built using ReactJS and Tailwind CSS.

## To Run

```
1. clone repo
2. cd taskify
3. npm install
4. npm run dev
```

## Live Link
[taskifybynazeef.netlify.app](https://taskifybynazeef.netlify.app/)

## Technologies Used

* ReactJS
* TailwindCSS
* Vite
